# FocusForest — API Reference

**Base URL:** `/api/v1`  
**Auth:** All endpoints require a valid Supabase JWT delivered via `httpOnly` cookie (`sb-access-token`) unless marked `[public]`.  
**Versioning:** `/api/v1` prefix on every route. No exceptions.

## 1. Auth

### `POST /auth/signup` [public]

Create a new account via Supabase Auth.

**Request body (Zod schema):**
```ts
z.object({
  email: z.string().email(),
  password: z.string().min(8),
  name: z.string().min(1).max(50),
  utcOffset: z.number().int().min(-720).max(840).default(0),
})
```

**Response `200`:**
```json
{
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "name": "Alice",
    "avatarUrl": null,
    "utcOffset": 330
  }
}
```
> Sets `sb-access-token` and `sb-refresh-token` as `httpOnly` cookies.

**Errors:** `400` validation, `409` email already registered.

---

### `POST /auth/login` [public]

Login with email and password.

**Request body (Zod schema):**
```ts
z.object({
  email: z.string().email(),
  password: z.string().min(1),
})
```

**Response `200`:**
```json
{
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "name": "Alice",
    "avatarUrl": null
  }
}
```
> Sets `sb-access-token` and `sb-refresh-token` as `httpOnly` cookies.

**Errors:** `400` validation, `401` invalid credentials.

---

### `POST /auth/logout`

Invalidate the current session.

**Response `200`:**
```json
{ "message": "Logged out" }
```
> Clears both auth cookies.

---

### `GET /auth/me`

Get the current authenticated user's profile.

**Response `200`:**
```json
{
  "id": "uuid",
  "email": "user@example.com",
  "name": "Alice",
  "avatarUrl": "https://...",
  "utcOffset": 330,
  "isPrivate": false,
  "createdAt": "2025-03-01T00:00:00.000Z"
}
```

---

### `PATCH /auth/profile`

Update the current authenticated user's profile settings.

**Request body (Zod schema):**
```ts
z.object({
  name: z.string().min(1).max(50).optional(),
  avatarUrl: z.string().nullable().optional(),
  isPrivate: z.boolean().optional(),
})
```

**Request example:**
```json
{
  "name": "Alice",
  "isPrivate": true
}
```

**Response `200`:**
```json
{
  "id": "uuid",
  "email": "user@example.com",
  "name": "Alice",
  "avatarUrl": null,
  "utcOffset": 330,
  "isPrivate": true,
  "createdAt": "2025-03-01T00:00:00.000Z"
}
```

> **Privacy Note:** When `isPrivate` is set to `true`, the user will be excluded from the global solo leaderboard. Setting it to `false` (default) makes the user visible on the global leaderboard. Privacy settings do not affect group leaderboards.

**Errors:** `400` validation, `401` unauthorized, `404` user not found.

## 2. Sessions

### `POST /sessions`

Submit a completed focus session. The score engine runs **server-side only** — the client never sends or receives `stageProgress` directly.

**Request body (Zod schema):**
```ts
z.object({
  variant: z.enum(["sprint", "classic", "deep_work", "flow", "custom"]),
  focusMinutes: z.number().int().min(1).max(240),
  taskText: z.string().max(200).optional(),
  taskStatus: z.enum(["completed", "carried", "none"]).default("none"),
  clientSessionId: z.string().uuid(), // generated by client, used for deduplication
})
```

**Server logic (do not skip):**
```
taskMultiplier = taskStatus === "completed" ? 1.5 : 1.0
stageProgress  = (focusMinutes / 25) × taskMultiplier
```

**Response `200`:**
```json
{
  "tree": {
    "stage": 2,
    "glowLevel": 3,
    "stageProgress": 2.5,
    "totalSessions": 3,
    "sessionsWithTask": 2
  },
  "streak": {
    "currentStreak": 5
  }
}
```

**Errors:** `400` validation, `409` duplicate `clientSessionId`.

---

### `POST /sessions/start`

Start an immersive mode session with live tracking.

**Request body (Zod schema):**
```ts
z.object({
  variant: z.enum(["sprint", "classic", "deep_work", "flow", "custom"]),
  focusMinutes: z.number().int().min(1).max(240),
  taskText: z.string().max(200).optional(),
  clientSessionId: z.string().uuid(),
})
```

**Response `201`:**
```json
{
  "sessionId": "uuid",
  "expectedEndAt": "2025-03-10T15:25:00.000Z"
}
```

**Errors:** `400` validation, `401` unauthorized, `409` duplicate `clientSessionId`.

---

### `POST /sessions/:id/complete`

Complete an active immersive mode session. Validates that at least 80% of the expected focus time has elapsed.

**Request body (Zod schema):**
```ts
z.object({
  taskStatus: z.enum(["completed", "carried", "none"]),
})
```

**Response `200`:**
```json
{
  "tree": {
    "stage": 2,
    "glowLevel": 3,
    "stageProgress": 2.5,
    "totalSessions": 3,
    "sessionsWithTask": 2
  },
  "streak": {
    "currentStreak": 5
  }
}
```

**Errors:** 
- `400` validation
- `400 SESSION_TOO_SHORT` — elapsed time < 80% of focusMinutes
- `400 SESSION_NOT_ACTIVE` — session already completed or abandoned
- `403` not session owner
- `404` session not found

---

### `POST /sessions/:id/abandon`

Abandon an active immersive mode session without completing it.

**Request body:** None

**Response `200`:**
```json
{
  "message": "Session abandoned successfully."
}
```

**Errors:**
- `400 SESSION_NOT_ACTIVE` — session already completed or abandoned
- `403` not session owner
- `404` session not found

---

### `GET /sessions`

Get session history for the authenticated user.

**Query params:**
| Param | Type | Description |
|-------|------|-------------|
| `startDate` | `YYYY-MM-DD` | Optional. Filter from this date (inclusive). |
| `endDate` | `YYYY-MM-DD` | Optional. Filter to this date (inclusive). |
| `limit` | `number` | Optional. Default `50`, max `200`. |
| `offset` | `number` | Optional. Default `0`. |

**Response `200`:**
```json
{
  "sessions": [
    {
      "id": "uuid",
      "variant": "classic",
      "focusMinutes": 25,
      "taskText": "Finish Chapter 4",
      "taskStatus": "completed",
      "stageProgress": 1.5,
      "createdAt": "2025-03-10T14:30:00.000Z"
    }
  ],
  "total": 42
}
```

## 3. Trees & Calendar

### `GET /trees/today`

Get today's live tree state for the authenticated user. Uses the user's `utcOffset` to determine "today".

**Response `200`:**
```json
{
  "date": "2025-03-10",
  "stage": 2,
  "glowLevel": 2,
  "totalSessions": 2,
  "sessionsWithTask": 2,
  "isBare": false,
  "finalisedAt": null
}
```
> Returns `stage: 0, isBare: false` if no sessions yet today (seed state).

---

### `GET /trees/calendar`

Get all daily trees for the authenticated user.

**Query params:**
| Param | Type | Description |
|-------|------|-------------|
| `month` | `1–12` | Optional. Filter by month. |
| `year` | `YYYY` | Optional. Filter by year. Required if `month` is set. |

**Response `200`:**
```json
{
  "trees": [
    {
      "date": "2025-03-10",
      "stage": 4,
      "glowLevel": 4,
      "totalSessions": 4,
      "sessionsWithTask": 4,
      "isBare": false,
      "finalisedAt": "2025-03-11T00:00:00.000Z"
    }
  ]
}
```

---

### `GET /trees/week/:weekId`

Get all 7 day slots for a specific week.

**Params:** `weekId` — format `YYYY-Www` (e.g. `2025-W10`).

**Response `200`:**
```json
{
  "weekId": "2025-W10",
  "startDate": "2025-03-03",
  "endDate": "2025-03-09",
  "complete": true,
  "days": [
    {
      "date": "2025-03-03",
      "stage": 4,
      "glowLevel": 3,
      "isBare": false
    }
  ]
}
```

## 4. Groups

### `POST /groups`

Create a new group. The authenticated user becomes the admin.

**Request body (Zod schema):**
```ts
z.object({
  name: z.string().min(1).max(50),
})
```

**Response `201`:**
```json
{
  "id": "uuid",
  "name": "Study Squad",
  "inviteCode": "XK7P2Q",
  "memberCount": 1,
  "createdAt": "2025-03-10T00:00:00.000Z"
}
```

---

### `POST /groups/join`

Join a group via its 6-digit invite code.

**Request body (Zod schema):**
```ts
z.object({
  inviteCode: z.string().length(6).toUpperCase(),
})
```

**Response `200`:**
```json
{
  "id": "uuid",
  "name": "Study Squad",
  "memberCount": 3
}
```

**Errors:** `404` code not found, `409` already a member, `403` group is full (5 members).

---

### `GET /groups/:id`

Get group details, member list, and forest stats.

**Response `200`:**
```json
{
  "id": "uuid",
  "name": "Study Squad",
  "inviteCode": "XK7P2Q",
  "memberCount": 3,
  "adminUserId": "uuid",

    {
      "userId": "uuid",
```

**Errors:** `404` group not found, `403` not a member.

### `GET /groups/:id/calendar`

Get the group's collective daily output — accountability view.

**Response `200`:**
```json
{

      "date": "2025-03-10",
```

---

### `DELETE /groups/:id/members/:userId`

Remove a member (admin only) or leave the group (self).

**Response `200`:**
```json
{ "message": "Member removed" }
```

**Errors:** `403` not admin (when removing others), `404` member not found.

## 5. Leaderboard

### `GET /leaderboard/solo`

Solo rankings by total all-time completed trees.

**Query params:**
| Param | Type | Description |
|-------|------|-------------|
| `scope` | `global \| none` | `global` = all public users, `none` = empty leaderboard. Default `global`. |
| `page` | `number` | Default `1`. |
| `limit` | `number` | Default `20`, max `50`. |

**Response `200`:**
```json
{
  "scope": "global",
  "page": 1,
  "total": 500,
  "entries": [
    {
      "rank": 1,
      "userId": "uuid",
      "name": "Alice",
      "avatarUrl": null,
      "totalTrees": 120,
      "currentStreak": 30
    }
  ]
}
```

> **Privacy Note:** Users who have set `isPrivate: true` in their profile are excluded from the global leaderboard. Only public users (`isPrivate: false`) appear in the results. Privacy settings do not affect group leaderboards.

**Errors:** `400` invalid scope value, `401` unauthorized.

---

### `GET /leaderboard/groups`

Group rankings by total group forest size.

**Query params:** Same as `/leaderboard/solo`.

**Response `200`:**
```json
{
  "scope": "global",
  "page": 1,
  "total": 100,
  "entries": [
    {
      "rank": 1,
      "groupId": "uuid",
      "name": "Study Squad",
      "totalTrees": 320,
      "memberCount": 5
    }
  ]
}
```

## 6. Common Types

### TimerVariant
```ts
type TimerVariant = "sprint" | "classic" | "deep_work" | "flow" | "custom"
```

| Variant | Focus Minutes | Stage Progress (base) |
|---------|-------------|----------------------|
| `sprint` | 15 | 0.6 |
| `classic` | 25 | 1.0 |
| `deep_work` | 50 | 2.0 |
| `flow` | 90 | 3.6 |
| `custom` | user-defined | `focusMinutes / 25` |

### TaskStatus
```ts
type TaskStatus = "completed" | "carried" | "none"
```

| Status | Task Multiplier | Description |
|--------|----------------|-------------|
| `completed` | 1.5 | User marked task done — golden glow |
| `carried` | 1.0 | Task carried to next session |
| `none` | 1.0 | No task was set |

### TreeStage
```ts
type TreeStage = 0 | 1 | 2 | 3 | 4

// 0 = Seed    (0 points)
// 1 = Sprout  (1.0 cumulative points)
// 2 = Sapling (2.0 cumulative points)
// 3 = Young Tree (3.0 cumulative points)
// 4 = Full Tree  (4.0 cumulative points)
```

### GlowLevel
```ts
type GlowLevel = 0 | 1 | 2 | 3 | 4
// Computed: Math.floor((sessionsWithTask / totalSessions) * 4)
// 4 = full golden glow, 0 = plain tree
```
